# HireVue

HireVue ist ein KI-gestütztes Tool für Video-Interviews und Bewerberauswahl im Recruiting-Prozess.

- **Einsatzbereiche:** Video-Interviews, Bewerberauswahl
- **Besonderheiten:** KI-gestützte Analyse, Zeitersparnis
